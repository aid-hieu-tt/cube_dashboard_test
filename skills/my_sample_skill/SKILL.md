# Hướng dẫn chi tiết (Detailed Instructions)

## Mục đích của Skill
Mô tả rõ ràng khi nào và tại sao nên sử dụng skill này. Antigravity sẽ đọc file này để biết cách thực hiện một tác vụ cụ thể.

## Các bước thực hiện

### Bước 1: Phân tích
- Hướng dẫn AI cách phân tích công việc bắt đầu từ đâu.

### Bước 2: Thực thi
- Cung cấp các lệnh cần chạy.
- Liệt kê các file cần chỉnh sửa.

### Bước 3: Kiểm tra
- Cách để xác minh rằng công việc đã được hoàn thành đúng.

## Các thư mục đính kèm
- `scripts/`: Chứa các file script hỗ trợ (Bash, Python, Node.js...) nếu cần.
- `examples/`: Chứa các mẫu code, file ví dụ để tham khảo.
- `resources/`: Chứa các file bổ sung, templates, cấu hình chuẩn.
